Worked on several PHP projects in the gaming industry( private Silkroad MMO servers http://www.xtremetop100.com/silkroad-online, http://www.maximizedphantasy.com/ ) providing web development for specific game related actions ( register, password recovery, unstuck, donations, marketplace etc ). Provided both PHP and UI development using CSS and Javascript (jQuery).

Websites were hosted on different servers being linked to the game server through a PHP REST webservice. Due to performance issues we decided using simultaneously 2 database solutions:


  • Microsoft SQL Server (due to the fact that the game was running on it )
  • MySQL storing data not bound to the game server ( necesary for website only - marketplace items, logs, internationalization and localization).

Through these projects I self-taught web development patterns ( Singleton, Object pool, Abstract factory, Lazy initialization, Front Controller, MVC, SOA and others), Beside this I learned to communicate with various people around the world who were contributing to these private server, mostly from the arabic area.

I have built an announcement board in PHP for the college institution front entrance, containing a slideshare of announcements made by college officials and the current schedual for all classrooms in the facility. Moreover I have added a GIF built map containing steps from the entrance to each room in the building. Through an administration page college officials with a designated username and password were able to change scheduled classes and add announcements.
