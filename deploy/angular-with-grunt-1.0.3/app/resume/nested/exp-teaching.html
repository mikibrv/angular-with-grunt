web development teacher

october-december 2014

  I have created and teached a web development course for 2nd year students of Transilvania University, Computer Science faculty.

Course consisted of:

  • Understanding the WEB / HTTP
  • Web Security (OWASP top 10 with live examples)
  • HTML(5) / CSS(3)
  • Mobile-First Responsive Web Design
  • Twitter Bootstrap
  • Javascript (namespace / modules / prototype)
  • jQuery - good practices when developing jQuery plugins
  • Facebook API and Google Maps API usage
Course length: 10 weeks
Attendance (min/max): 38-65 split in 2 parallel groups

After the course the best students were offered jobs in the company.
