Alt Brasov - Innovation and technology cluster

November, 23rd 2014

  Sustained a workshop on using Apache Camel to implement Enterprise Integration Patterns for real time data analysis using a master slave distribution.
  I have asked every participant to download the code and open a bat file, starting a maven exec plugin.They have all connected to my laptop acting as a master which load balanced tweets to them from Twitter Stream API for further analysis.

  I have proved the power of parallel analysis and also of MongoDB aggregation framework to speed up processes.   Communication was done using ActiveMQ and Spring Remoting (remote bean invokation).

  Technologies used: Apache Camel Spring WebSockets Jetty ActiveMQ MongoDB JSON

While in Paris, I attended several international meetups about AngularJS, BigData and Apache Cassandra.

I enjoyed the opportunity of having a chat with Cassandra's CTO and other influent people in software development.

Apache Cassandra

After building a PoC with Cassandra for a customer and participating at a Cassandra meetup in Paris I have sustained an internal KS demonstrating the ease of use and the power of Cassandra when working with a cluster handling large amount of data.
